## Getting Started

### Prerequisites

#### Version

|  Nvim  | Zsh | Kitty  | Tmux | Hyprland |
| ------ | --- | -----  | ---- | -------- |
|  0.10  | 5.9 | 0.31.0 | 3.4  |  0.45.2  |

### Installation

> For wayland setup, please refer to the official doc [here](https://hyprland.org/)

1. Clone the repo

   ```sh
   git clone https://github.com/Naleeee/Dotfiles.git
   ```

2. Run the auto installer script for a full setup

   ```sh
   ./scripts/AutoInstaller.sh
   ```
